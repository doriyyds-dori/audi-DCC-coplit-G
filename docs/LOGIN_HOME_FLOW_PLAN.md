# 智慧外呼助手第一阶段登录与首页交互方案

> 日期：2026-03-24

---

## 1. 第一阶段登录目标

**通过用户名密码识别使用者身份，将后续通话记录准确归属到具体人员和门店。**

---

## 2. 登录页最小能力

| 能力 | 说明 |
|------|------|
| 用户名输入 | 必填 |
| 密码输入 | 必填，密码遮挡 |
| 登录按钮 | 校验通过后跳转 |
| 错误提示 | 用户名或密码错误时提示"用户名或密码错误"；账号被禁用时提示"账号已被禁用，请联系管理员" |

**不做**：注册、找回密码、短信验证码、第三方登录、记住密码。

---

## 3. 登录成功后的最小跳转逻辑

| 角色 | 登录后跳转 | 原因 |
|------|-----------|------|
| 普通用户 | 话术助手页 | 普通用户唯一的使用场景就是打电话用话术，直接进入 |
| 超级管理员 | 管理后台（今日统计页） | 管理员最高频的需求是看当天数据 |

超级管理员在管理后台可通过入口切换到话术助手页。

---

## 4. 普通用户首页最小结构

普通用户首页就是**现有的话术助手页面**，在现有基础上只增加两处：

| 区域 | 作用 | 备注 |
|------|------|------|
| 顶部工具栏（已有） | 场景切换、上传/下载、帮助、重置 | 复用现有 HeaderToolbar |
| **顶部右侧新增**：用户信息 | 显示当前用户姓名 + 登出按钮 | 新增 |
| 左侧对话面板（已有） | 通话模拟 | 复用现有 ChatPanel |
| 右侧 FAQ 面板（已有） | FAQ 查询 | 复用现有 FaqPanel |
| 右下角当前阶段（已有） | 步骤提示 | 复用现有 CurrentStepInfo |
| **通话状态弹窗**（新增） | 通话结束后弹出，选择状态后记录 | 新增 |

**不做**：个人统计、消息通知、快捷入口、工作台。

---

## 5. 超级管理员首页最小结构

管理后台首页为"今日统计"，整体结构：

| 区域 | 内容 |
|------|------|
| 顶部导航 | 品牌标识、菜单切换（今日统计 / 用户管理 / 门店管理）、"进入话术助手"入口、登出 |
| 日期选择 | 默认今天，可切换到其他日期 |
| 门店筛选 | 下拉选择门店，默认"全部门店" |
| 统计表格 | 列：用户姓名、门店、总通话数、未接通、无意向、有意向 |

**不做**：图表可视化、导出 Excel、趋势分析、对比分析。

---

## 6. 通话记录的最小触发流程

1. 普通用户登录后，进入话术助手页
2. 选择客户类型（保有潜客 / 首次邀约）
3. 点击「开始邀约」，进入通话模拟
4. 使用话术辅助应对客户，过程中可使用 FAQ 菜单
5. 通话过程中或结束后，用户主动点击「结束并记录通话」按钮
6. 弹出通话状态选择弹窗：未接通 / 明确无意向 / 有意向
7. 用户必须选择一个状态，弹窗才会关闭
8. 选择后，系统将 `user_id` + `store_id` + `status` + `scenario_type` + `call_date` + `created_at` 写入服务端，对话区重置，可开始下一通

**重要区分**：
- 「重置对话」= 清空当前对话重新演练，**不产生通话记录**
- 「结束并记录通话」= 本次通话结束，**触发状态选择并写入记录**

---

## 7. 路由与访问控制的最小规则

| 路由 | 需要登录 | 普通用户可见 | 管理员可见 |
|------|---------|------------|-----------|
| `/login` | 否 | 是 | 是 |
| `/app`（话术助手） | 是 | 是 | 是 |
| `/admin`（管理后台） | 是 | 否 | 是 |
| `/admin/users`（用户管理） | 是 | 否 | 是 |
| `/admin/stores`（门店管理） | 是 | 否 | 是 |

**访问控制规则**：
- 未登录访问任何需登录页面 → 跳转 `/login`
- 普通用户访问 `/admin` → 跳转 `/app`
- 已登录用户访问 `/login` → 跳转到对应默认首页

---

## 8. 第一阶段明确不做的交互能力

| 不做 | 原因 |
|------|------|
| 注册页面 | 用户由管理员创建 |
| 找回密码 | 400 人规模由管理员重置即可 |
| 通知中心 | 第一阶段无推送需求 |
| 多组织切换 | 第一阶段只有一层门店结构 |
| 个人中心/设置 | 无个性化需求 |
| 普通用户查看自身统计 | 第一阶段统计仅管理端可见 |
| 页面内引导/新手教程 | 已有 HelpModal 使用说明 |

---

## 9. 登录与首页交互结论

1. **登录只做最简单的用户名密码登录**，不做注册、验证码、第三方登录
2. **普通用户首页就是现有话术助手页面**，只新增用户信息显示和通话状态弹窗
3. **管理员首页是今日统计表格**，通过顶部菜单切换到用户管理和门店管理
4. **通话记录通过状态弹窗触发写入**，用户必须选择状态才算完成一次通话闭环
5. **路由控制只分两层**：需不需要登录、是不是管理员
